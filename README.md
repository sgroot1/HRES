# Helios Setup Manager

Setup Manager is a race engineering plugin for Helios.

## Features

- Session Management
- Run Management
- Setup Sheets
- Start / End Setup
- Change History
- Autosave
- Weather Integration (planned)
- Telemetry Linking (planned)

## Development

Install dependencies

```bash
npm install
```

Run

```bash
npm run dev
```

Build

```bash
npm run build
```
